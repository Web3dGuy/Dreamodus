# DreaModus Software Inc. — Landing Page

A single-page landing site, built with Node.js + Express.

## Run

```bash
npm install
npm start          # http://localhost:3000
npm run dev        # auto-restart on changes
```

Set a custom port with `PORT=8080 npm start`.

## Structure

```
server.js            Express server (static hosting)
public/index.html    The landing page
public/styles.css    Brand-themed styles (teal → blue → purple)
Dreamodus-*.jpg      Brand logos, served at /brand/...
```
